# VOID RUNNER — Asteroid Dodge 🚀

## Description
A browser-based mini game where the player controls a spaceship and avoids asteroids while collecting energy orbs.

## Controls
- Mouse / Arrow keys → Move
- Avoid asteroids
- Collect orbs for points

## Win Condition
Reach **Level 5** and **3000 score** to win.

## Features
- Start screen
- Game loop
- Score system
- Levels
- Win & Game Over screens
- Restart + Main Menu
