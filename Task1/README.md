# TEDx CUSAT — Landing Page 🎤

## 📌 Overview

This project is a **single-page landing website** for a fictional **TEDx CUSAT event**.
It is designed to give users essential information about the event and encourage them to register.

The page follows modern UI principles with a **clean layout, smooth scrolling, and responsive design**.

---

## 🚀 Features

### 🏠 Hero Section

* Event name: **TEDx CUSAT**
* Tagline: *Ideas worth spreading*
* Event details (Date, Venue, Seats)
* Call-to-action buttons:

  * Get Tickets
  * Meet the Speakers

---

### ℹ️ About Section

* Explains what **TEDx** is
* Highlights the purpose of the event
* Includes visual statistic cards (speakers, attendees)

---

### 🎤 Speakers Section

* Grid layout of speaker cards
* Includes:

  * Name
  * Role
  * Topic
* Placeholder data used for demonstration

---

### 🎟 Tickets / Registration Section

* Different ticket tiers:

  * Student
  * General
  * VIP
  * Online (Free)
* Simple registration form (Name + Email)

---

### 📱 Responsive Design

* Fully responsive layout using **CSS media queries**
* Works across:

  * Desktop
  * Tablet
  * Mobile

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* Google Fonts (Bebas Neue, DM Sans)

---

## 🎯 Objective

This project was created as part of **Task 1**, which required:

* A hero section
* About section
* Speaker/lineup section
* Ticket/registration call-to-action
* Basic mobile responsiveness

All requirements have been successfully implemented.
